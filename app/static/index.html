Pipeline Control

Follow these 3 steps to populate your universe and start seeing results

First time? Start here.

Your database is empty until you run Step 1. Step 1 syncs all 895 stocks from your Supabase equity_universe table (takes ~30 sec). Step 2 scores them all deterministically (instant). Step 3 runs the AI agents on any individual stock from the Analyze tab.

1

Sync Equity Universe

Reads all 895 stocks directly from your Supabase equity_universe table — no yfinance calls needed. Financial ratios (ROE, revenue CAGR, P/E, P/B, D/E) are mapped automatically.

Takes ~30 sec. Run again anytime to pick up new stocks added to equity_universe.

2

Run Tier-1 Prefilter

Scores every tracked stock using deterministic rules: revenue growth ≥ 12%, market cap check, P/E, P/B, ROE, and leverage. No AI cost — pure Python.

Completes in seconds. After this, the Universe and Dashboard will show rankings.

3

Add a Custom Stock

Add any NSE/BSE stock outside the Nifty 50. Use the NSE ticker with .NS suffix (e.g. ZOMATO.NS, DMART.NS).

Then go to the Analyze tab to run the full 4-agent AI pipeline on it.
